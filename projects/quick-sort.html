Past Projects: Quick Sort

ICS4U0: Sorting

For this project, we had to choose a particular sorting algorithm, and make a demonstration program. I completed this project with the help of my friend Min Kang, and we decided to choose quick sort. Quick Sort is a very popular divide and conquer sorting algorithim, and for a more indepth explanation click here. While coding the project, we were given a choice of 3 libraries to use: Processing, hsa, or Java swing. We initally started the project in Java swing, but found many difficulties and so we migrated to Processing. I'd say this is probably my most visually pleasing programs. I was inspired by a vaporwave aesthetic, and used a miami vice colour palette. You can find this project on my Github, or by clicking the Github button at the bottom of this page.
