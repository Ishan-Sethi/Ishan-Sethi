ICS3U3: Final Project

Grade 11 ICS was mainly based on teaching Java concepts through a program called Ready to Program Java, using a package called hsa. Using hsa was considerable more difficult, and more frustrating then Processing. Processing handled many of the more difficult tasks for you, such as the draw loop and mouse input. hsa had none of that, its graphics were a considerable downgrade from Processing, and there was always this weird flicker in the background. All my griping aside, for my final project, I was given the task of making the classic board game Snakes and Ladders. You can find this project on my Github, or by clicking the Github button at the bottom of this page.
