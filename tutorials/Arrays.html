Arrays

What is an Array?

An array is a type of data structure in Computer Programming, that holds a collection of data. Different coding languages handle arrays in different ways, but this tutorial will mainly be about how arrays work in Java. Every element in an array must be off the same type, either a primitive data type (char, int, double, etc.), or an object (String, Colors, etc.). When you first declare an array, every value in the array is set to it's default value. Doubles, ints and chars are set to 0, booleans are set to false, and Strings are any objects are set to null. Here is syntax on how to declare, and initialize an array:

int[] intArr; // Declaring a new integer Array
intArr = new int[4]; // Initializes integer array of length 4
String[] objectArr = new String[4]; // Declaring, and initializing an array in the same line

There are two ways to store data in an array, you can set up the entire array when initializing it, or access each individual element and change it. Every value in an array is stored at a particular index, starting at 0 and going to the length of the array minus one. This can be somewhat confusing to new programmers, as the first element in every array is stored at index 0, but we humans normally start counting from 1. Here is an example of how to put data in an array:

String[] names = {"Ishan", "Ms. Krasteva"}; // Automatically sets length to 2
int[] intArr new int[4]; // Initializes integer array of length 4
intArr[0] = 2; // Sets the value at index 0 (first element) to be 2
intArr[1] = -180; // Sets the value at index 1 (second element) to be -180
